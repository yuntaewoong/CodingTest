# Dynamic Programming
Dynamic Programming(동적계획법)은 해답인 값을 찾기위해  
더 작은단계의 해답의 값을 이용하는 것이다.  
즉 재활용이다.  

# 예시
- 피보나치에 DP를 적용하면, 
```
D[i] = D[i-2] + D[i-1]
D[0] = 1
``` 
이 점화식을 이용해서 D배열을 채우면  
상당히 많은 값들을 재사용할수 있다.(재귀 시간복잡도는 O(N!),DP시간복잡도는 O(N))  
하지만, 이런 점화식은 쉽게 생각할 수 없다는 단점이 있다.  
장점은 엄청 빠르다는 것이다.
